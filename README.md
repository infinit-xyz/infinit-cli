# @infinit-xyz/cli

[![codecov](https://codecov.io/gh/melvinmcrn/infinit-cli/graph/badge.svg?token=RY39X0ESZ8)](https://codecov.io/gh/melvinmcrn/infinit-cli)


To install dependencies:

```bash
bun install
```

To build CLI:

```bash
bun run dev
```

To run CLI:
```bash
bunx infinit
```
